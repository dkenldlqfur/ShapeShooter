🎯 ShapeShooter 게임 매뉴얼

게임 소개

ShapeShooter는 3인칭 숄더뷰 시점에서 회전하는 3D 도형의 모든 면을 맞춰 색을 바꾸는 슈팅 퍼즐 게임입니다.

플레이어는 원점에 떠 있는 도형 주위를 궤도 이동하며 총알을 발사합니다. 도형의 모든 면이 흰색(HP 0)이 되면 스테이지를 클리어하게 됩니다.

조작법

이동

입력 동작
W / ↑ 위로 궤도 이동
S / ↓ 아래로 궤도 이동
A / ← 왼쪽으로 궤도 이동
D / → 오른쪽으로 궤도 이동
Left Shift 가속 이동 (1.2배)
💡 참고 플레이어는 원점(도형 위치)을 중심으로 구면 위를 이동합니다. 항상 도형을 바라보는 방향을 유지합니다.

발사

입력 동작
Space 총알 발사
마우스 좌클릭 총알 발사
💡 참고 총알은 항상 원점(도형 중심) 방향으로 발사됩니다.

게임 규칙

기본 흐름

  1. 시작 화면 — 시작 버튼을 클릭하여 게임 시작
  2. 카운트다운 — "3, 2, 1, 시작!" 카운트다운 후 스테이지 시작
  3. 플레이 — 도형의 모든 면을 맞춰 HP를 0으로 만들기
  4. 스테이지 클리어 — 모든 면이 완료되면 다음 스테이지로 진행
  5. 게임 클리어 — 모든 스테이지를 클리어하면 게임 완료

면(Face) 시스템

각 면은 HP를 가지고 있으며, 총알이 맞을 때마다 1씩 감소합니다.

HP 색상
6🟣 보라
5🔵 파랑
4🟢 초록
3🟡 노랑
2🟠 주황
1🔴 빨강
0⬜ 흰색 (완료!)
  • HP 0인 면을 다시 맞추면 HP가 1로 복원됩니다 (빨간색으로 되돌아감). 주의하세요!
  • 링크된 면이 있는 경우, 한 면을 맞추면 연결된 면들도 함께 변경됩니다.

충돌 판정

  • 총알은 면의 바깥쪽에서 안쪽으로 충돌할 때만 유효합니다.
  • 도형 내부에서 밖으로 통과하는 총알은 무시됩니다.

회전 패턴

스테이지가 진행될수록 도형의 회전 난이도가 올라갑니다.

패턴 설명
Static 회전 없음 (튜토리얼용)
SingleAxis X, Y, Z 중 하나의 축으로 고정 회전
MultiAxis 랜덤 조합 축으로 고정 회전
ReactiveAxis MultiAxis 기반이지만, 총알이 맞을 때마다 회전 축이 바뀜
Random 2~5초마다 랜덤 축으로 자동 변경

UI 안내

플레이 중 표시 정보

  • Stage — 현재 스테이지 번호
  • 시간 — 현재 스테이지 경과 시간 (실시간 갱신)
  • 탄환 — 현재 스테이지에서 발사한 총알 수

기록 표시

  • 최고 시간 — 해당 스테이지의 최단 클리어 시간
  • 최소 탄환 — 해당 스테이지의 최소 발사 횟수
💡 참고 기록은 자동으로 저장되며, 더 빠른 시간으로 클리어하면 갱신됩니다.

공략 팁

  1. 면 위치 파악이 우선 — 발사 전 도형을 관찰하여 남은 면의 위치를 확인하세요.
  2. HP 0인 면 주의 — 이미 흰색인 면을 맞추면 HP가 복원됩니다. 정확히 조준하세요.
  3. 가속 이동 활용 — Shift로 빠르게 이동하여 최적의 위치를 선점하세요.
