# ShapeShooter 게임 매뉴얼

## 게임 소개

**ShapeShooter**는 3인칭 숄더뷰 시점에서 회전하는 3D 다면체(Polygon)의 모든 면을 맞춰 색을 바꾸는 캐주얼 퍼즐 슈팅 게임입니다.

플레이어는 화면 중앙 원점에 떠 있는 입체 도형 주위를 자유롭게 궤도 비행하며 총알을 쏘아 맞춥니다. 도형을 구성하는 모든 면의 체력(HP)을 0(흰색)으로 깎아내면 스테이지를 클리어하게 됩니다.

---

## 🕹️ 조작법

### 1) 이동
| 입력 | 동작 |
|------|------|
| `W` / `↑` | 위로 궤도 이동 |
| `S` / `↓` | 아래로 궤도 이동 |
| `A` / `←` | 왼쪽으로 궤도 이동 |
| `D` / `→` | 오른쪽으로 궤도 이동 |
| `Left Shift` | 가속 이동 (1.2배 부스트) |

> 플레이어는 원점(도형 위치)을 중심으로 구면 위를 미끄러지듯 이동합니다. 카메라는 항상 중앙 타겟 도형을 바라보며 짐벌락(Gimbal Lock) 없이 자연스럽게 회전합니다.

### 2) 발사
| 입력 | 동작 |
|------|------|
| `Space` | 총알 발사 |
| `마우스 좌클릭` | 마우스 커서가 가리키는 화면 위치를 향해 정밀 사격 |

> 마우스 클릭 사격을 이용하면 3D 공간 상에서 원하는 특정 폴리곤(면)을 정확하게 저격할 수 있습니다.

---

##  규칙 및 승리 조건 (Gameplay Loop)

1. **시작 화면** — 게임 시작 버튼 클릭.
2. **카운트다운** — "3, 2, 1, 시작!" 알림과 동시에 전투 돌입.
3. **타격 및 색상 변화** — 도형의 표면을 맞추면 HP가 1씩 차감되며 타격된 면의 색상이 변합니다.
4. **스테이지 클리어** — 도형의 **모든 면**의 HP가 0이 되어 완전히 하얗게 변색하면 클리어됩니다.

### 🎨 면(Polygon) 체력 시스템
각 폴리곤은 기본적으로 생명력을 상징하는 고유의 색상을 띱니다. 총알에 피격될 때마다 그라데이션 애니메이션(EaseOut)과 함께 색이 한 단계씩 떨어집니다.

| HP | 표시 색상 |
|----|------|
| **6** | 보라 |
| **5** | 파랑 |
| **4** | 초록 |
| **3** | 노랑 |
| **2** | 주황 |
| **1** | 빨강 |
| **0** | **흰색 (완료!)** |

> ⚠️ **주의 (페널티)**
> 이미 HP가 0이 되어 하얗게 정화된 면을 **다시 쏘아 맞출 경우**, 그 면의 HP가 다시 1(빨간색)로 복원됩니다. 총알을 낭비하지 않도록 궤도를 도는 위치와 화력을 신중히 조절해야 합니다.

## 🔄 보스 패턴 (Rotation Types)

스테이지가 올라갈수록 표적 도형의 방어(회전) 패턴이 악랄해집니다.

| 패턴 이름 | 작동 방식 |
|------|------|
| **Static** | 멈춰있는 고정 타겟 (튜토리얼 난이도). |
| **SingleAxis** | X, Y, Z 규칙적인 단일 축을 기준으로 천천히 자전합니다. |
| **MultiAxis** | 무작위로 복합된 축을 기준으로 빙글빙글 돕니다 (예측 사격 필요). |
| **ReactiveAxis** | 플레이어의 **총알이 표면에 적중할 때마다** 회전 방향(기저 축)이 순간적으로 무작위 비틀립니다. |
| **Random** | 2~5초마다 주행 방향을 알 수 없는 무작위 축으로 급선회합니다. |

## 🏆 성적 산출 (UI Records)

게임 도중 상단 UI를 통해 실시간 기량을 확인할 수 있으며, 스테이지 클리어 시 최단 기록이 로컬에 영구 저장됩니다.

- **종합 소요 시간 (Timer)**: 얼마나 빠르게 클리어했는지 측정.
- **총 사용 탄환 (Bullets)**: 탄낭비를 줄이고 얼마나 예리하게 명중시켰는지를 측정.
- **베스트 스코어 (Best Record)**: 최고 기록 산출 시스템을 통해 무한한 타임어택/스코어링 플레이를 장려합니다.
